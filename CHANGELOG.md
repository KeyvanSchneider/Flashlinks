# Changelog

## 0.1.0

Initial release of Flashlinks.

- Create flashcards from `::label::` links.
- Edit front and back content.
- Review cards with spaced repetition metadata.
- Flip cards with click or Space.
- Open the source note from a flashcard.
- Organize cards in virtual folders.
- Drag and drop cards between folders.
- Render Obsidian image embeds inside cards.
- Export cards to Markdown.
- Export cards to an Anki-compatible tab-separated text file.
- Switch the interface between English and French.
